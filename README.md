# Banten Pedia - Balinese Upakara Collection

Banten Pedia is a form of concern for Balinese tangible culture: Upakara. With Banten Pedia, we store all of our knowledge about Upakara in a written form, make it easier for the new generation to learn about upakara. Our elder, even us will not stay forever in the world. It's our era to contribute to our own culture in a modern way.

Banten Pedia relies on a chain system. Bigger upakara `A` has a list of its constituents: upakara `B`, `C`, and `D`. The smaller upakara is arranged to form a larger upakara.

## Getting Started

### Summary

Banten Pedia is a monorepo project. The backend built using [Go](https://go.dev), a robust backend language and the frontend built using [SvelteKit](https://svelte.dev/docs/kit/introduction), a compiler-based frontend framework.

## Contributing

Contributions are divided into two types: upakara and development.

### Upakara

For contributors who knows about Balinese upakara:

1. Access [TBA](https://example.com) then create an account.
2. Start contribution by proposing new upakara or update existing upakara. `Penyarikan` will review it then approve if it is according to the upakara rules.
3. If you have more knowledge about Balinese upakara, please email us [TBA](https://example.com) and ask for an account upgrade to `Penyarikan` with supporting evidence.
4. `Penyarikan` has an access to approve upakara proposal.

### Development

Developers can contribute by adding new requested features, bug fixing, or improving the documentation. Please read [CONTRIBUTING.md](./CONTRIBUTING.md) for further information.
